# 회사 정산 서버리스 앱

서버 없이 GitHub Pages에서 바로 쓰는 회사 영수증/분기 예산 정리 앱입니다.

## 사용 방식

GitHub Pages 링크를 iPhone Safari에서 열어 사용합니다.

- 영수증 입력/수정/삭제
- 월별 영수증 합계 확인
- 회사 제출용 `.xlsx` 파일 생성
- 분기별 지급금/예산/실사용/잔여금 관리
- 자주 쓰는 사용내역/사용처 빠른 입력
- 백업 JSON 다운로드/가져오기

## 데이터 저장 위치

서버와 DB를 쓰지 않습니다. 데이터는 현재 브라우저의 `localStorage`에 저장됩니다.

즉, iPhone Safari에서 입력한 데이터는 그 iPhone Safari 안에 저장됩니다.

주의할 점:

- 다른 기기에서는 자동으로 보이지 않습니다.
- Safari 방문 기록/웹사이트 데이터 삭제 시 사라질 수 있습니다.
- 기기 변경 전에는 반드시 `백업` 탭에서 JSON 파일을 내려받아 보관하세요.

## 엑셀 저장

`영수증` 탭에서 월을 선택한 뒤 `엑셀 저장`을 누르면 브라우저가 직접 `.xlsx` 파일을 생성합니다.

생성되는 시트:

- `Expenses List_법인`

엑셀에는 결제일자 열을 넣지 않습니다.

## 로컬에서 열기

서버 없이 파일만 열어도 됩니다.

```text
docs/index.html
```

GitHub Pages에서는 `docs/` 폴더가 사이트 루트가 됩니다.
